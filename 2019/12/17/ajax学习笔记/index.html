Asynchronous Javascript And XML (异步的javascript 和 XML)

发送Ajax请求
服务器处理完成后返回响应，通过javascript对响应进行处理

jquery的$.ajax()方法使用：
1、传入的时一个json对象，用于描述发送的细节$.ajax(json);
2、常用的json对象设置：
url 发送请求地址；
type请求类型get/post;
data向服务器传递的参数
dataType服务器响应的数据类型text/json/xml/html/jsonp/script
success 接收响应时的处理函数
error 请求失败时的处理函数
